`tiff-filter` is a code for denoising and generating point clouds from tomography tiff files.

### Installation 

```
git clone https://github.com/badw/tiff-filter.git 
cd tiff-filter 
pip install . 
```

### Examples 

an example Jupyter Notebook can be found here:
